# Inserting Documents in a MongoDB Collection
To insert documents into a MongoDB collection, you can use the `insertOne()` or `insertMany()` methods. The `insertOne()` method inserts a single document, while the `insertMany()` method inserts multiple documents into the collection.

Here's an example of inserting a single document using the `insertOne()` method:

## Insert One Document

```javascript

db.grades.insertOne({
  student_id: 654321,
  products: [
    {
      type: "exam",
      score: 90,
    },
    {
      type: "homework",
      score: 59,
    },
    {
      type: "quiz",
      score: 75,
    },
    {
      type: "homework",
      score: 88,
    },
  ],
  class_id: 550,
})

```
